# Portfolio

Personal portfolio website for Eshan Rajapakshe

**Live Site:** [https://eshanrajapakshe.netlify.app/](https://eshanrajapakshe.netlify.app/)

## Tech Stack

- React 18
- TypeScript
- SCSS
- React Icons

## Features

- Responsive design
- Scroll-based animations
- CV preview modal
- NPM packages showcase
- CodePen embeds
- Project portfolio

## Development

```bash
npm install
npm start
```

## Build

```bash
npm run build
```
